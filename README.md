# ProEngineerCalc

Professional engineering calculator toolbox website built with Next.js.

## Features

- **HVAC Calculators**: Dew point, psychrometric properties, duct velocity, cooling load, pump head
- **Electrical Calculators**: Power, voltage drop, cable ampacity, illuminance, battery capacity
- **Fluid Mechanics**: Pipe pressure loss, flow conversion, Reynolds number
- **Structural Engineering**: Concrete mix, beam load, unit converter

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- CSS Modules
- PWA Support
- Vercel Deployment

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

Deploy to Vercel with automatic GitHub integration.
